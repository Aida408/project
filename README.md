


# сервис для сегментации клиентов банка

## О проекте

Этот проект сочетает в себе анализ данных, обучение модели машинного обучения и веб-приложение для классификации новых клиентов в реальном времени. Цель проекта — помочь банкам лучше понимать клиентов, оптимизировать маркетинговые стратегии, персонализировать продукты и управлять рисками. В проекте применяются методы кластеризация и создается интерактивный инструмент, который можно использовать в реальной бизнес-среде.

---

## Функциональные возможности

* Анализ и визуализация клиентских данных кредитных карт.
* Кластеризация клиентов с помощью K-Means.
* Автоматическое определение сегмента нового клиента.
* Интерактивное веб-приложение на Streamlit для ввода данных и отображения результатов.
* Подробное описание каждого сегмента с ключевыми характеристиками.

---

## Данные

* **Источник:** [Kaggle: Credit Card Dataset for Clustering](https://www.kaggle.com/datasets/arjunbhasin2013/ccdata)
* **Объём:** ~9000 клиентов, 18 поведенческих метрик (баланс, частота и сумма покупок, использование кредитного лимита и др.)
* **Особенности:** Данные требуют предварительной обработки, включая заполнение пропусков и масштабирование признаков для корректной работы K-Means.


## Основные этапы работы

### 1. Анализ данных и подготовка модели

* Исследование данных и визуализация ключевых метрик.
* Обработка пропусков и масштабирование признаков с помощью `StandardScaler`.
* Определение оптимального числа кластеров (метод локтя и коэффициент силуэта).
* Обучение модели **K-Means** и сохранение модели и скейлера.

### 2. Веб-приложение на Streamlit

* Создание интерфейса для ввода данных нового клиента (`st.slider`, `st.number_input`).
* Загрузка модели и скейлера с кешированием:
* Масштабирование данных и определение сегмента клиента с выводом описания на экран.

### 3. Управление проектом

* Контроль версий через **Git**.
* Публикация проекта на **GitHub** с полным кодом, моделями и документацией.

## Технологии

* **Язык:** Python 3.x
* **Анализ данных:** Pandas, NumPy

* **Веб-приложение:** Streamlit
